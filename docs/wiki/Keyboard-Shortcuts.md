# Keyboard Shortcuts

## File / batch

| Shortcut | Action |
| --- | --- |
| `⌘O` | Open folder |
| `⇧⌘A` | Apply ALL Stuff (smart batch) |
| `⌘R` | Apply to Selection (file-batch) |
| `⌘A` | Select all in current section |
| `Space` | Toggle Mark on selected row(s) |
| `⌫` | Remove selected row(s) from list (does not touch disk) |

## Stabilize / Reference

| Shortcut | Action |
| --- | --- |
| `R` | Pin / unpin current row as Reference frame (gold star) |

## Preview

| Shortcut | Action |
| --- | --- |
| `B` | Compare side panel — toggle source-SER thumb vs. current file |
| `C` | Highlight clipped pixels (per-channel ≥ 99.5 %) red over the preview — diagnostic-only, saved files unaffected |
| `A` | Display auto-range (per-texture auto-stretch + pow midtone) |
| `⌘=` (or `⌘+`) | Zoom in 25 % |
| `⌘-` | Zoom out 25 % |
| `⌘0` | Fit to window |
| `⌘1` | 1:1 (one image pixel = one drawable pixel) |
| `⌘2` | 200 % |
| Double-click | Reset zoom to fit |
| Mouse drag | Hand-tool pan (image follows cursor in both axes) |
| ⌥ + scroll | Anchored zoom around cursor |
| Trackpad pinch | Zoom around cursor |

## Player (Memory + SER scrub)

| Shortcut | Action |
| --- | --- |
| `P` | Play / Pause |
| `◀` | Previous frame |
| `▶` | Next frame |

## Section switching

| Shortcut | Action |
| --- | --- |
