# Keyboard Shortcuts

## File / batch

| Shortcut | Action |
| --- | --- |
| `⌘O` | Open folder |
| `⇧⌘A` | Apply ALL Stuff (smart batch) |
| `⌘R` | Apply to Selection (file-batch) |
| `⌘A` | Select all in current section |
| `Space` | Toggle Mark on selected row(s) |
| `⌫` | Remove selected row(s) from list (does not touch disk) |

## Stabilize / Reference

| Shortcut | Action |
| --- | --- |
| `R` | Pin / unpin current row as Reference frame (gold star) |

## Preview

| Shortcut | Action |
| --- | --- |
| `B` | Before / After toggle |
| `⌘=` (or `⌘+`) | Zoom in 25 % |
| `⌘-` | Zoom out 25 % |
| `⌘0` | Fit to window |
| `⌘1` | 1:1 (one image pixel = one drawable pixel) |
| `⌘2` | 200 % |
| Double-click | Reset zoom to fit |
| Mouse-drag horizontal | Photoshop-style zoom |
| Trackpad pinch | Zoom around cursor |
| Scroll wheel (zoomed in) | Pan |

## Player (Memory + SER scrub)

| Shortcut | Action |
| --- | --- |
| `P` | Play / Pause |
| `◀` | Previous frame |
| `▶` | Next frame |

## Section switching

| Shortcut | Action |
| --- | --- |
| Click section toggle bar | Inputs / Memory / Outputs |

## Help / About

Available from the application menu — no global shortcut.
